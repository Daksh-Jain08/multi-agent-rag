Use "outdated" when some documents contain older or superseded information. Use "misinformation" when some documents appear false or unreliable.

Next, decide whether you should abstain from answering. Set should_abstain to true if the documents do not provide enough information, if the evidence is contradictory and cannot be resolved, or if the information is unreliable. Otherwise, set it to false.

If you do not abstain, provide a concise final answer based strictly on the documents. Do not add any external knowledge or speculation. If you abstain, return a short message indicating that the evidence is insufficient or conflicting.

You must follow these constraints strictly. The output must be valid JSON and must contain exactly three keys: "conflict_type", "should_abstain", and "final_answer". Do not include explanations, reasoning, analysis, or any extra fields. Do not output markdown or any text outside the JSON. Do not hallucinate or infer beyond what is supported by the documents.

Output format:

{
  "conflict_type": "...",
  "should_abstain": true/false,
  "final_answer": "..."
}
